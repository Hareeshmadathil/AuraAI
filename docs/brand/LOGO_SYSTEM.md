# AuraAI Logo System

All current marks are **CONCEPT — FOUNDER REVIEW REQUIRED**. No winner has been
selected and trademark clearance is outstanding.

## Concept A — Aura Orbit

Open asymmetric arcs coordinate around a stable core. Strengths are flow,
balance, and dashboard presence. Ring geometry carries moderate similarity
risk, so the open asymmetric construction must be retained and screened.

## Concept B — A Monogram

Connected operational paths build an abstract A. It is compact and strong at
favicon size. The inner path must remain visible, and the geometry requires
similarity screening before adoption.

## Concept C — Signal Core

A stable core coordinates offset directional rails. It is modular and maps
well to system-state interfaces. The offset rails must remain distinct from a
generic play button or sunburst.

## Usage rules

- Full wordmark: minimum 120 CSS pixels wide.
- Compact mark: minimum 32 CSS pixels wide.
- App icon: minimum 24 CSS pixels; always provide an accessible label.
- Clear space: at least one core-width around every mark.
- Light backgrounds: use a deep-ink monochrome treatment.
- Dark backgrounds: use light neutral or Aura Aqua treatment.
- Favicon/app icon: use one concept’s compact icon, never the wordmark.

Each family is designed to retain its silhouette in one color. Dashboard and
social-avatar suitability notes appear on `/brand`. Founder selection should
be followed by professional legal/trademark screening and small-size testing.
