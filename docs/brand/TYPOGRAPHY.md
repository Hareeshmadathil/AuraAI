# AuraAI Typography

AuraAI uses local system fonts only.

- Display and headings: Segoe UI Variable Display, Segoe UI, system UI.
- Body and labels: Segoe UI Variable Text, Segoe UI, system UI.
- Code and identifiers: Cascadia Code, SFMono-Regular, Consolas, monospace.
- KPIs: display stack with tabular numerals.

Page titles use a compact line height and responsive scale. Section and card
headings prioritize hierarchy over decoration. Labels and badges never drop
below the documented extra-small token. Body copy uses a readable line length
and relaxed line height; tables retain at least the small text size.

Potential future hosted fonts may be evaluated only with privacy, performance,
licensing, and offline fallbacks addressed. No hosted font is approved today.
