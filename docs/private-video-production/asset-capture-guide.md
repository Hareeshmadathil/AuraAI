# Founder asset capture guide

Use 1920×1080 at 30 fps where available. Keep the desktop clean, disable
notifications, hide bookmarks and private tabs, and capture only the evidence
named in the generated manifest. Never expose `.env`, API keys, personal email,
recovery codes, private customer information, or unrelated personal files.

Required Mission Zero opportunities include early transcript code, Git history,
the dashboard, mission terminal, roster, Creative Quality breakdown, the
43-character subtitle failure and correction, current test results, the founder
gate, and relevant public profiles.

Place founder inputs under the prepared package's ignored `founder-assets/`
directory using the exact manifest filenames. The repository-level
`founder_assets/` path is also ignored for future reusable inputs. Use only
supplied or demonstrably licensed footage, images, music, and effects.
