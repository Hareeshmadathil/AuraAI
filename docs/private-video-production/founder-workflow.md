# Founder workflow

1. Run the environment check and inspect the installed local voice list.
2. Prepare the approved Mission Zero package. This creates plans only.
3. Record the requested screen captures using the generated capture manifest.
4. Select one installed voice and create a short audition.
5. Review pronunciation and pacing. An audition does not approve content.
6. Explicitly confirm content approval before full narration generation.
7. Inspect the timeline, subtitles, placeholders, and missing-asset report.
8. Explicitly confirm private-render approval before rendering.
9. Review the verified local MP4 and choose approve-private, request-edit, or
   reject-private.

Private-draft approval never uploads or publishes. Publishing approval is not
available in this milestone.
