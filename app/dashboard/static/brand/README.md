# AuraAI Brand Assets

These local SVGs are original concept proposals for founder review. None is
an approved AuraAI logo, trademark clearance has not been performed, and no
asset should be used publicly until the founder selects a direction and legal
similarity screening is complete.

The icon sprite supports the local dashboard. Logo, wordmark, and app-icon
families A, B, and C are described in `docs/brand/LOGO_SYSTEM.md` and indexed
by `manifest.json`. Assets contain no raster data, external fonts, scripts, or
remote references.
