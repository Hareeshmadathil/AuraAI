# AURA AI

## Vision

AURA AI is an AI-powered content automation platform.

The goal is to transform one long-form video into multiple viral short-form videos automatically.

## Planned AI Agents

- Video Downloader
- Audio Extractor
- Speech-to-Text
- Content Analyzer
- Viral Clip Finder
- Video Editor
- Caption Generator
- Thumbnail Generator
- SEO Generator
- Publishing Agent
